Contributing

FlexiList is a funny and very new project, so there are a lot of things that can be improved.

Inspired on jQuery plugins like DataTables and jTable, we needed something like those, but for Angular.JS

FL has the unique flexibility advantage provided by the Angular.JS template engine so it is not constrained to be just a table.

If you liked FL, you can be part of it in any way you like, just to name a few:

  • Provide any feedback.
  • Bugfixes and code improvements (certainly many things can be done in a better way).
  • Tests (yes, there are no tests, shame on me).
  • Documentation. A lot of things can be better explained.
  • New and awesome templates.
  • Sample server side implementations. (for PHP, ASP, Node.js, etc).
  • Even this web.
  • Whatever else...

In summary: all types of contributions to make FL (a little or a lot) better are welcome.

May the Force be with you!
