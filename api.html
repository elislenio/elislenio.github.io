API reference

Options

Use the $scope.list.options model to properly configure FL.

 

Available options are:

data (type: array; default: false)

Use this option to load inline data directly from the options scope.

Usage example

jsonFile (type: string; default: false)

Use this option to specify a JSON file containing the dataset to be loaded.

listURL (type: string; default: false)

Use this option to specify an URL to retrieve the dataset to be loaded.

method (type: string; default: GET)

Sets the preferred method (GET, POST) for server requests. This option does not take effect if listURL is set to false.

urlencoded (type: boolean; default: true)

If enabled, query string is sent urlencoded using jQuery.param(). This option takes effect if listURL is set to true and method is set to GET.

autoload (type: boolean; default: true)

When disabled, you must explicitly call loadData to get the list populated.

selectable (type: boolean; default: true)

Enables or disables row selection.

multiselect (type: boolean; default: true)

Enables or disables multiple row selection. This option does not take effect if selectable is set to false.

limit (type: boolean; default: true)

Use this option to limit the result set.

where (type: array; default: empty array)

Use this option to filter the result set. See filters.

searchOnClient (type: boolean; default: false)

If enabled, searching is processed on client side. This option does not take effect if listURL is set to false.

sortable (type: boolean; default: true)

Enables or disables sorting on the result set.

orderby (type: array; default: empty array)

Use this option to sort the result set.

sortOnClient (type: boolean; default: false)

If enabled, sorting is processed on client side. This option does not take effect if listURL is set to false.

pagination (type: boolean; default: true)

Enables or disables pagination on the result set.

paginationOnClient (type: boolean; default: false)

If enabled, pagination is processed on client side. This option does not take effect if listURL is set to false.

pagination_clear_selection (type: boolean; default: true)

If disabled, selected rows are not cleared on page changes. This option does not take effect if pagination is set to false.

pagesize (type: integer; default: 10)

Sets the number of rows per page. This option does not take effect if pagination is set to false.

pages (type: integer; default: 5)

Sets the number of pages to show on the pagination widget. This option does not take effect if pagination is set to false.

log (type: object)

Use this option to control the console output.
Usage example: 
log: {id: 'MyList', err: true, debug: false}

Scope extensions

$scope.list.loadData type function(options) returns: void

Triggers data loading.

$scope.list.change type function(options) returns: void

Change options without reloading data.

$scope.list.refresh type function() returns: void

Reloads data using the current options settings.

$scope.$emit('flStartOp', object)

Dispatches an event when FL starts processing.

object: {op: 'opname'}

$scope.$emit('flComplete', object)

Dispatches an event when FL completes processing.

object: {result: 'ERROR', message: 'reason'} on error or {result: 'OK'} on success

$scope.list.selectEnabled type function() returns: boolean

Returns the select enabled status.

$scope.list.multiselectEnabled type function() returns: boolean

Returns the multiple selection enabled status.

$scope.list.paginationEnabled type function() returns: boolean

Returns pagination enabled status.

$scope.list.sortEnabled type function() returns: boolean

Returns sortable enabled status.

$scope.list.searchEnabled type function() returns: boolean

Returns search enabled status.

$scope.list.records type array

The records array contains all records in current page.

$scope.list.isEmpty type function() returns: boolean

Returns true whenever the result set has no records.

$scope.list.isSortedAsc type function(field) returns: boolean

Returns true whenever the list is sorted asc by field.
This function is not available if sortable option is set to false.

$scope.list.isSortedDesc type function(field) returns: boolean

Returns true whenever the list is sorted desc by field.
This function is not available if sortable option is set to false.

$scope.list.isRecordSelected type function(record) returns: boolean

Returns true whenever the given record is selected.
This function is not available if selectable option is set to false.

$scope.list.recordToggleSelect type function(record) returns: void

Toggles row selection.
This function is not available if selectable option is set to false.

$scope.list.enforceSelection type function(record) returns: void

Triggers the validation of row selection after changing record.flSelected model.
This function is not available if selectable option is set to false.

$scope.list.getToggleSelNextState type function() returns: boolean

Returns the next selection state for all rows.
This function is not available if selectable option is set to false.

$scope.list.toggleSelectAll type function() returns: boolean

Toggles the selection state for all rows.
This function is not available if selectable option is set to false.

$scope.list.getSelectedCount type function() returns: boolean

Returns the selected rows count.
This function is not available if selectable option is set to false.

$scope.list.getSelectedRecords type function() returns: array

Returns the selected records array.
This function is not available if selectable option is set to false.

$scope.list.pagination_info type object

The pagination_info object contains these properties:
  • firstrec: number of the first record in the current page.
  • lastrec: number of the last record in the current page.
  • rowcount: number of records in the working dataset.
  • currpage: number of the current page in the working dataset.
  • lastpage: number of the last page in the working dataset.
  • totalpages: number of pages in the working dataset.
  • pagesize: number of records per page.

$scope.list.changePage type function(pagenum) returns: void

Triggers page change.
This function is not available if pagination option is set to false.

$scope.list.setPageSize type function(size) returns: void

Changes the page size and switches to first page.
This function is not available if pagination option is set to false.

$scope.list.server_response type object

Stores the server response.
This model is not available if listURL option is set to false.

Filters

FL comes with several filters. You can add your own custom filters.

The provided filters only applies to client side processing and you may implement them on server side.

"eq": Evaluates the Equal condition.

Set option to true for case insensitive search.

"ne": Evaluates the Not Equal condition.

Set option to true for case insensitive search.

"gt": Evaluates the Greater condition.

"ge": Evaluates the Greater Equal condition.

"lt": Evaluates the Lower condition.

"le": Evaluates the Lower Equal condition.

"like_l": Evaluates the Like condition with a wildcard on the left.

Set option to true for case insensitive search.

"like_r": Evaluates the Like condition with a wildcard on the right.

Set option to true for case insensitive search.

"like_b": Evaluates the Like condition with a wildcard on both the left and right.

Set option to true for case insensitive search.

"is_null": Evaluates the Is Null condition.

"is_not_null": Evaluates the Is Not Null condition.

"regexp": Evaluates the a condition based on a regular expression.

Set option to apply a RegExp modifier.
