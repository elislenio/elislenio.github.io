Search

You can filter the dataset by setting the where option.

Load and search a JSON file

Live demo:

The searchPhonesCtrl controller sets up the FL options.

The searchExample directive implements our template. The partial of our template is search-tpl.html

Our template displays a very basic search form, the searchExCtrl controller extends $scope.list by adding the $scope.list.search function to trigger the search on the dataset.

The $scope.list.change API function applies the new where clause without reloading the dataset.

The $scope.list.loadData API function requests a dataset reload with the new where clause.

Load and search data from server side

If where is present in the server response, FL will use this value to ensure that the view accurately reflects the server response, in spite of the requested value.

Filters

FL comes with several filters that you can directly use, when jsonFile option is active. (See the API reference for details)

Also, you can extend FL with your own additional filters.

When using FL with server side processing, filters must be implemented with the technology of your choice.
