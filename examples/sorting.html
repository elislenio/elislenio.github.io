Sort data

You can sort the dataset by setting the orderby option.

Load and sort a JSON file

Live demo:

The sortPhonesCtrl controller sets up the FL options.

The sortExample directive implements our template. The partial of our template is sort-tpl.html

The sortExCtrl controller extends $scope.list by adding the $scope.list.toggleSorting custom functionality to our template.

The $scope.list.change API function applies the new orderby clause without reloading the dataset.

The $scope.list.loadData API function requests a dataset reload with the new orderby clause.

Load and sort data from server side

If orderby is present in the server response, FL will use this value to ensure that the view accurately reflects the server response, in spite of the requested value.

The sortPhonesServerCtrl controller sets up the list options.

The sortExample directive implements our template.

The sortServerExCtrl controller extends $scope.list by adding the $scope.list.toggleSorting custom functionality to our template and using the $scope.list.loadData API function.

The partial of our template is sort-tpl.html
