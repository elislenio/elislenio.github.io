Pagination

These examples uses the Angular UI Bootstrap pagination, this is not a restriction and any other UI component can be used alternatively.

Pagination is enabled by default and can be disabled by setting the pagination option to false.

Pagination on client

Use the pagesize option to set the number of records per page.

Use the pages option to set the number of pages shown in the pagination bar.

Pagination on server

If pagesize is present in the server response, FL will use this value to ensure that the view accurately reflects the server response, in spite of the requested value. (The same applies to offset)
